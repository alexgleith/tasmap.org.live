    Welcome to TasMap.org

    What is this?

    A simple, fast, lightweight mapping system. It discovers available datasets from Tasmanian government authorities and provides viewing and feature information retrieval.

    Data sources

    Currently supporting Esri REST services from:

    • The LIST — Land Information System Tasmania

    Who built this?

    Originally built by Alex Leith for GovHack 2016, rebuilt with MapLibre GL JS.
